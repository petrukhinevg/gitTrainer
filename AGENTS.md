# Инструкции для агента репозитория

Этот файл нужен только для жёстких repo-wide инвариантов. Подробный bootstrap и маршрутизация живут в `LOCAL_AGENT_START.md`.

## Старт нового чата

0. Всегда отвечай и работай только на русском языке.
1. Следуй маршрутизации из `LOCAL_AGENT_START.md` и открывай только нужные документы. `LOCAL_AGENT_START.md` - точка ввода в проект.
2. Возвращайся в этот файл только за repo-wide инвариантами.

## Repo-wide инварианты

- Не коммить `LOCAL_AGENT_START.md` и `.env`.
