# Дорожная карта

## Продукт

- SPA для практики Git через короткие упражнения.
- MVP: каталог, экран упражнения, проверка ответа, прогресс.

## Правила

- Здесь только product-level `parent issues`.
- Детализация реализации живёт в side-specific roadmap и workflow docs.
- Если крупного изменения нет в roadmap, сначала обнови этот файл.

## MVP

1. Выбор упражнения из каталога.
2. Прохождение упражнения в рабочем пространстве.
3. Проверка ответа с объяснением.
4. Прогресс и подсказка следующего шага.

## Parent issues

### Родительская задача 1. MVP просмотра каталога сценариев

Цель: дать пользователю каталог упражнений и выбор сценария.

Включает:

- summary-данные сценариев
- публикацию каталога
- список, фильтры, состояния загрузки и ошибок

### Родительская задача 2. MVP контекста упражнения и рабочего пространства

Цель: дать пользователю понять задачу, состояние репозитория и ожидаемый результат.

Включает:

- detail payload упражнения
- инструкции и контекст репозитория
- shell рабочего пространства

### Родительская задача 3. MVP отправки ответа и проверки корректности

Цель: дать пользователю отправить ответ и сразу получить результат проверки.

Включает:

- запуск сессии
- отправку ответа
- machine-readable feedback

### Родительская задача 4. MVP управляемых повторных попыток и подсказок

Цель: помочь пользователю восстановиться после ошибки, не теряя контекст.

Включает:

- объяснения ошибок
- сохранение контекста между попытками
- progressive hints

### Родительская задача 5. MVP прогресса и подсказок по следующему шагу

Цель: сделать повторную практику заметной и подсказать, что делать дальше.

Включает:

- результаты попыток и completion state
- отображение решённого и незавершённого
- рекомендации следующих упражнений
